日志功能应以所追踪事件级别或严重性而定。各级别适用性如下（以严重性递增）
｜级别｜何时使用｜

| 级别      | 何时使用                       |
| --------- | ------------------------------ |
| `DEBUG`   | 细节信息，仅当诊断问题时适用。 |
| `INFO`    | 确认程序按预期运行。           |
| `WARNING` | 表明程序可能发生异常           |
| `ERROR`          |                      有严重的问题，程序部分功能已经无法运行          |
默认的级别是 `WARNING`，意味着只会追踪该级别及以上的事件，除非更改日志配置。
